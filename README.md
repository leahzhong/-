# 贪吃蛇游戏

一个使用原生HTML/CSS/JavaScript开发的网页版贪吃蛇游戏。

## 功能特点
- 经典贪吃蛇玩法
- 键盘方向键控制
- 实时计分系统
- 简洁优雅的Apple风格设计
- 黄色渐变的蛇身，绿色苹果形状的食物
- 吃到食物时的粒子爆炸特效
- 吃到食物和游戏结束的音效
- 最高分本地存储
- 无需安装任何依赖,直接在浏览器中运行

## 如何运行
双击打开 `index.html` 文件即可在浏览器中运行游戏。

## 操作说明
- 使用键盘方向键(↑↓←→)控制蛇的移动方向
- 吃到食物得分并增长身体
- 撞到墙壁或自己的身体会游戏结束
- 点击"重新开始"按钮可以开始新游戏
